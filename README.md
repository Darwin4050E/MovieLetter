# MovieLetter

MovieLetter es una aplicación que permite descubrir, valorar y gestionar películas de manera fácil e intuitiva. Con una interfaz amigable, la aplicación permite a los usuarios acceder a información detallada sobre películas, sin necesidad de iniciar sesión.

## Funcionalidades principales

- *Perfil personal sin inicio de sesión:* Crea un perfil donde puedes interactuar con la plataforma sin necesidad de registrarte.
- *Valoración y reseñas:* Los usuarios pueden valorar las películas con estrellas y dejar reseñas para compartir sus opiniones.
- *Listas personalizadas:* Agrega películas a tu lista de "Pendientes" o "Favoritos" y quítalas cuando ya las hayas visto o cambiado de opinión.
- *Información detallada de películas:* Accede a sinopsis, trailers, calificaciones y más, gracias a la integración con una API externa.

## Tecnologías utilizadas

- *Frontend:* React + Typescript, Ionic, Vite, MUI y Capacitor.
- *Backend:* La app consume la API de TMDB (https://www.themoviedb.org/) desde el cliente.
